# ML Mini Projects

This repository contains three small machine learning projects built with Python. Each project highlights a different technique in the ML workflow — from regression and clustering to data visualization.

## Projects

1. **Car Sales Prediction (Regression)**
   - Uses Linear Regression to predict car sales based on features.
   - Includes feature scaling, training, prediction, and evaluation with RMSE and R² score.

2. **Mall Customer Segmentation (Clustering)**
   - Applies K-Means clustering to group mall customers.
   - Demonstrates the Elbow Method to find the optimal number of clusters.

3. **Temperature vs Humidity Visualization**
   - Creates scatter plots to explore the relationship between temperature and humidity.
   - Useful for understanding correlations in environmental data.

## Features
- Data preprocessing (scaling)
- Model training and prediction
- Evaluation metrics (RMSE, R², inertia)
- Visualizations (scatter plots, elbow curve)
